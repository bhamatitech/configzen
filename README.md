# configzen 🧘

The simplest way to handle configuration in Python. 

- **Zero Config:** Just point to a file and go.
- **Dot-Access:** `config.db.port` instead of `config["db"]["port"]`.
- **Env Overrides:** Automatically override file values with `APP_KEY=value`.
- **Type Smart:** Automatically casts Env strings to the correct type.

## Installation
```bash
pip install configzen
# For YAML support:
pip install configzen[yaml]
```
